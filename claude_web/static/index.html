Agent Loop
让 Claude 自动迭代：执行目标、跑测试、失败重试，直到完成、阻塞或预算用完。
Loop 会在后端持续运行并复用当前权限策略和工作目录；每轮失败最多自动重试 2 次，连续遇到相同测试失败会暂停。需要中断时点输入框上方的“停止”。

    使用技巧
    • 粘贴图片直接发送（支持截图）
    • 粘贴 URL 自动抓取网页正文作为上下文
    • 拖拽文档（PDF / DOCX / XLSX 等）上传并提取文本
    • 点击输入框工具栏的循环按钮可启动 Agent Loop，让 Claude 按目标自动迭代到完成、阻塞或预算用完
    • 手机访问建议使用同 WiFi + 指定本机 IP + 访问码；手机端是问答模式，可提问、看结果、切换已保存项目，管理功能请回到电脑本机操作
    • 手机上左侧会话列表是抽屉式导航，点遮罩 / 关闭按钮 / 选中会话都会收起；底部输入区已适配 iPhone 安全区
    • 窄屏下代码块、工具结果和 Edit Diff 会自动横向滚动或上下排列，避免长代码撑开页面
    • 点击输入框工具栏的魔法棒，基于黄金样本和个人规则优化当前 Prompt
    • Chrome 插件支持点击图标打开 Side Panel，也可读取当前页面全文或右键选中内容提问
    • 给关键用户消息或回答点 ⭐，再到左侧「标记」Tab 快速回看长会话里程碑
    • 给回答点 👍 或标记后，可作为 Prompt 优化器的黄金样本候选
    • 工具调用失败时可一键放行权限重试
    • Git 仓库自动 checkpoint，可一键回滚到对话前
    • 双击会话标题重命名，或点 🪄 让 AI 命名
